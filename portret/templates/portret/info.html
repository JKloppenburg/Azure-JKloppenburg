Info


Mijn naam is Jeanette Kloppenburg.
Als ZZP’er zoek ik een opdracht waarbij ik met data werk: Data Analist, BI Analist, Data Engineer, Data Specialist of DBA. Het grootste deel van mijn carrière heb ik gewerkt met een relationeel database management systeem gebaseerd op SQL. Ik ben ervaren in alle werkzaamheden die benodigd zijn om data te transformeren tot informatie en deze te presenteren in rapportages en dashboards. Ik beschik over de competenties om alle werkzaamheden in het hele traject uit te voeren en zal daarmee op dit gebied ook een waardevolle bijdrage kunnen leveren. Mijn ervaring heeft zich hoofdzakelijk ontwikkeld op het HP Nonstop (Tandem) platform. Het database management systeem op HP NonStop is relationeel en de gebruikte taal is SQL. In alle eerlijkheid kan ik zeggen dat ik niet de ervaring heb met alle tools die ingezet worden binnen ieder bedrijf. M.i. hoeft dat geen belemmering te zijn om snel aan de slag te kunnen. Mijn opgedane kennis, o.a. SQL, is makkelijk toepasbaar op ieder ander database management systeem wat gebaseerd is op SQL. Met mijn competenties kan ik de verschillende tools snel aanleren. Bij het aangaan van een nieuwe opdracht wil ik een inwerkperiode van 2 à 4 weken voor mijn rekening nemen. D.w.z. dat ik in die periode geen uren in rekening breng. Met wat inwerkhulp van collega’s en ook door mezelf te verdiepen in de materie en de gebruikte tools heb ik er alle vertrouwen in dat ik de opdracht naar volle tevredenheid van de klant zal uitvoeren.

Deze website

Deze website ben ik om twee redenen gaan bouwen: enerzijds als studieproject en anderzijds omdat ik denk er een beter beeld van mezelf mee te kunnen geven dan middels een C.V. op papier mogelijk is.
Nadat ik kennis heb opgedaan van Django, Python, HTML, CSS, JavaScript, MySQL, Azure, Git en GitHub, heb ik deze website zelf met deze tools gebouwd.
Ik ben begonnen met het hosten van de website in Azure. In Azure heb ik voor een Saas oplossing gekozen. De resources "Azure Database voor MySQL" en "Web-app" heb ik hiervoor gebruikt. Daarna heb ik me verdiept in PythonAnywhere waar op dit moment mijn site gehost wordt.
Ik ben trots op dit resultaat. Het toont m.i. ook aan dat ik mijn compenties goed kan aanwenden om, voor mij, nieuwe tools te gebruiken. Ik heb niet gekozen voor de makkelijkste oplossing om een website te bouwen omdat ik kennis wilde opdoen van meerdere talen en mogelijkheden. Zo geeft de C.V. tab van deze website meer informatie dan mijn C.V. op papier; Je kan klikken op mijn opleidingen om de diploma's of certificaten te zien. Als je met je muis over de functies gaat, dan vouwen de activiteiten uit. Als je op "getuigschrift" klikt, dan krijg je die, of een ander aan dat werk gereleerd document, te zien in een popup scherm. En als je op de LinkedIn verwijzing klikt, dan kom je in mijn LinkedIn pagina terecht. De meeste informatie die op de C.V. tab getoond wordt is opgeslagen in tabellen in MySQL.
